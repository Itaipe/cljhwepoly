Pour la gestion des statistiques :

--> La moyenne des examens effectués (en pourcentage) ne prend pas en compte les examens abandonnés.

--> Lorsque l'on abandonne un examen, le compteur des examens abandonnés est incrémenté.
L'énoncé stipule "L’abandon d’un examen apporte l’utilisateur à la page terminale (fin de l’examen) avec
une note de 0 / X". Nous avons considéré que X est le nombre de questions sélectionnés par le candidat
pour cet examen (et non le nombre de questions auxquelles il a déjà répondu).

--> Lorsqu'un examen n'est pas abandonné, il est considéré comme effectué (même si la note est de 0/X).

--> Le taux de réussite des examens est calculé par le nombre totale de questions répondues justement sur le nombre
totales de questions posées en examen. Ainsi, un examen sur 10 questions influencera plus le taux qu'un examen sur 5 questions.


Pour l'affichage des résultats des examens (lorsque l'on clique sur détails) :

--> Sont affichés dans les résultats des examens uniquement les examens effectués (ceux abandonnés ne sont pas affichés).

--> Le bouton détails n'est consultable que depuis le tableau de bord, tout comme le bouton remise à zéro, comme spécifié dans
l'énoncé des différents TP.
